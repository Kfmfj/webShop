# Web Shop for Local Store

A custom Django-based web shop built for a real offline business.
The system lets customers order products online while the store owner handles packaging and shipping manually.

# Project Overview

A fully dynamic e-commerce platform developed to help a local seller manage online orders efficiently.
The owner can:

Add, edit, or remove products anytime

Track customer orders

Manage payments and shipping manually

The goal was to connect the simplicity of a small shop with the flexibility of a modern web system.

# Tech Stack

Backend: Django (Python)

Frontend: HTML, CSS

Payment Gateway: Stripe

Features: Dynamic content, admin control, secure payments

# Key Features

Dynamic product catalog with admin management

Secure checkout using Stripe

Manual order fulfillment — seller receives payment and ships packages

Responsive UI for desktop and mobile

Protected user data and transactions
